Conway's Game of Life
Conway's Game of Life is a cellular automaton simulation developed by mathematician John Conway in 1970. This implementation provides an interactive graphical environment for exploring emergent behavior from simple rules.

Игра «Жизнь»
Игра «Жизнь» — это симуляция клеточного автомата, созданная математиком Джоном Конвеем в 1970 году. Данная реализация предоставляет интерактивную графическую среду для изучения эмерджентного поведения из простых правил.
